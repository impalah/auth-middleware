Azure Entra ID Infrastructure Setup
===================================

This guide walks you through setting up Azure Entra ID (formerly Azure Active Directory) infrastructure manually using the Azure Portal.

.. note::
   This documentation is currently under development. Please check back soon for detailed setup instructions.

Overview
========

Azure Entra ID setup will include:

- App Registration creation
- Service Principal configuration
- API permissions setup
- Group and user management
- Authentication flow configuration
